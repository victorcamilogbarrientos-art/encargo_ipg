Este proyecto es la aplicacion web solicitada desarrollada en react

Para iniciar debemos asegurarnos de tener instalado node.js

Se debe clonar el repositorio con el comando git clone [https://github.com/victorcamilogbarrientos-art/encargo_ipg.git](https://github.com/victorcamilogbarrientos-art/encargo_ipg.git)

Luego se debe entrar a la carpeta del proyecto con el comando [cd encargo-react-ipg]

Se intalan las dependencias con [npm install] para finalizar arracando la aplicacion con [npm run dev]

Para acceder a la aplicacion debera dar ctrl+click en el link que aparecera en la terminal
